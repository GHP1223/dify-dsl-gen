# Dify DSL Skill

Design, generate, review, refactor, and validate Dify Workflow, Chatflow, and RAG Pipeline DSL.

## Install

```
/plugin marketplace add GHP1223/dify-dsl-marketplace
```

## Usage

Mention Dify DSL, workflow, Chatflow, or RAG pipeline in your prompt. The skill auto-detects intent and routes to the correct workflow.

1.1.0 — Updated node references with real-world DSL format patterns from 87 production Dify examples.
